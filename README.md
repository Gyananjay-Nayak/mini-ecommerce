### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Gyananjay-Nayak/mini-ecommerce.git
   cd mini-ecommerce
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```
